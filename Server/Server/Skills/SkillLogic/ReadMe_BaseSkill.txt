注意：
	1、各门派公用技能的逻辑ID来自SkillLogic_T中的enum定义， Descriptor Formats 在各个技能的cpp文件里硬实现的。
	2、逻辑ID并非技能ID!!


技能列表：
ImpactsToTarget = 效果-->目标
LogicID : 0 ; 
效果描述部分的格式：|生效一次的附加效果1|引导周期的附加效果1|生效一次的附加效果2|引导周期的附加效果2|无...;

ImpactsToTargetAndSelf_T = 给目标和自己各仍几个效果
LogicID : 1 ; 
效果描述部分的格式：|生效一次的效果(自己)|生效一次的效果(目标)|引导周期的效果(自己)|引导周期的附加效果(目标)|无...;

TeleportAndImpacts = 瞬移目标到指定位置，并给目标加3个效果。
LogicID : 2 
效果描述部分的格式：|效果1|效果2|效果3|瞬移效果|无....； 
ArmorMastery = 防具专精
LogicID : 8 ; 
消耗部分的格式：|无.....
效果描述部分的格式：|防具的基础物理防御修正%|防具的基础魔法防御修正%|无...;

WeaponMastery = 武器专精
LogicID : 9 ; 
消耗部分的格式：|无.....
效果描述部分的格式：|武器的基础物理攻击修正%|武器的基础魔法攻击修正%|武器类型1|武器类型2|无...;
注: 
武器类型: 
	0: 大刀
	1: 长枪
	2: 单短
	3: 双短
	4: 扇
	5: 环
